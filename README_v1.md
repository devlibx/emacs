
### Golang debug
```shell

# Launch debug session with following - this will ask for config. Hit tab, and it will show 
# some configs - use config wiht File 
# Also make sure your buffer has to be your main.go file

M-x dap-debug 

# To get the debug control use
M-x dap-hydra
  - q => to quit
  - n, i, c -> these are some of the classic debug keys
```
